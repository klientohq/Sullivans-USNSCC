# The Sullivans USNSCC Website

Standalone static website prototype for The Sullivans Division of the U.S. Naval Sea Cadet Corps.

## Pages

- `index.html` - Home page
- `about.html` - About page
- `leadership.html` - Standalone Leadership page
- `resource-hub.html` - Standalone Resource Hub page
- `roadmap.html` - Standalone roadmap page
- `contact.html` - Contact page with current public info
- `store.html` - Store catalog preview with live WordPress product prices
- `cadet-dashboard.html` - Future login/dashboard placeholder

## Notes

- The current build is static and GitHub-ready.
- The store and checkout are planned for a later phase.
- The hero video is currently large and should be compressed before final deployment.
